Settings

Import from Google Sheet

Task in the first column, then one column per week headed Week 1, Week 2, … with that week's hours. (A single Hours column also works — same budget every week.) An optional Color column is honoured. Publish via File → Share → Publish to web as Comma-separated values (.csv), paste the link below. Import sets each week's budget, adds new tasks, and creates extra weeks if your sheet has more — it never deletes.

Sync (Supabase)

Enter the same three values on every device, or use the setup link.

SQL to run once in Supabase 
create table if not exists plans (
  id text primary key,
  data jsonb not null,
  updated_at timestamptz not null default now()
);
alter table plans enable row level security;
create policy "planner anon access" on plans
  for all using (true) with check (true);

Tips

Type a known task name to place it as a budgeted block; type any other name to drop in a one-off meeting (no hour budget). Days: every day (Mon–Sat), mon-fri, tue and thu, weekends (Sat). Drag either end of a block to resize; tap a block for nudge buttons. The 🗒 button shows/hides a sticky note you can drag anywhere (Ctrl+B bold, Ctrl+I italic, Ctrl+H pink).
